# UI/UX Pro Max 摘要（可用格式）

## 目的
將 ui-ux-pro-max-skill 轉換為本專案可直接使用的 UI/UX 指導摘要，便於後續持續優化版面與互動。

## 核心原則（摘要）
- 明確視覺方向：同一主題下，色彩、字體、陰影、圓角一致。
- 內容優先：題目可讀性 > 裝飾；層級清楚、焦點明確。
- 互動回饋：可點擊元素要有狀態、過渡、觸控感。
- 避免反模式：過度炫光、難辨識對比、無焦點提示。
- 行動端優先：單手操作、按鈕面積、避免滾動。
- 動態節奏：載入與切換以短動畫引導注意力。

## 本專案採用的方向
- 視覺主題：溫暖學習感、卡片層級、柔和陰影。
- 排版：標題醒目，內文字級對比明確。
- 動畫：題目與選項以短暫浮現動畫；彈窗以縮放進場。
- 觸控：按鈕與選項具按下回饋、避免誤觸。

## 套用位置
- `index.html`：色彩變數、按鈕狀態、動畫、彈窗、進度條。

## 後續擴充建議
- 若資料來源改為 DB/試算表，維持此視覺規範不變。
- 若要換主題，可先調整 CSS 變數與背景層。
